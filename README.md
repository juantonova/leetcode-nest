## Description
Это API, созданное с использованием Express.js, аналог Leetcode. Предназначено для управления задачами, комментариями, оценками, решениями и пользователями.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
